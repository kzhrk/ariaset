ariaset
===

ariaset make easier assign for aria-" properties.

## Usage

```js
import ariaset from 'ariaset';

const element = document.querySelector('.selector');

ariaset(element {
  label: 'description'
});
```

or 

```html
<script src="node_modules/ariaset/dist/ariaset.js">
<script>
const element = document.querySelector('.selector');

ariaset(element {
  label: 'description'
});
</script>
```

## Install

```
$ npm install ariaset
```
